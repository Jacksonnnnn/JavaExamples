()  - Parentheses - Used to contain parameters in a method definition (Also used for expressions)
{}  - Braces      - Used to contain the values of automatically initialized arrays.
[]  - Brackets    - Used to declare array types.
;   - Semicolon   - Terminates statements
,   - Comma       - Separates consecutive identifiers in a variable declaration.
.   - Period      - Used to separate package names from sub-packages and classes.
::  - Colons      - Used to create a method or constructor reference.
... - Ellipsis    - Used to create a method or constructor reference.
@   - Ampersand   - Begins an annotation.
